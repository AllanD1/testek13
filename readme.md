### Edite o aquivo .env

Adicione as chaves do Google reCaptcha 

`CAPTCHA_KEY= <CHAVE PUBLICA>`

`CAPTCHA_SECRET= <CHAVE PRIMARIA>`

Adicione a chave do [newsapi.org](https://newsapi.org)

`NEW_KEY= <Chave>`


Configuração Banco 

`DB_CONNECTION=mysql`

`DB_HOST=127.0.0.1`

`DB_PORT=3306`

`DB_DATABASE=<BANCO>`

`DB_USERNAME=<USUARIO>`

`DB_PASSWORD=<SENHA>`
